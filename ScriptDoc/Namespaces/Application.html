Application.Quit¶

Declaration¶

static void Application.Quit()

Quits the game immediately. Note: the game does not wait until the end of the current frame.

if self.quit_selected then
	Application.Quit()
end

Application.Sleep¶

Declaration¶

static void Application.Sleep(int milliseconds)

Suspends the application for a number of milliseconds.

if self.sleepy_time then
	Application.Sleep(10)
end

Application.GetFrame¶

Declaration¶

static int Application.GetFrame()

Returns the number of frames elapsed since the game launched. (Game begins on frame 0)

ReportError = function(error)
	Debug.LogError(error .. " occurred on frame " .. Application.GetFrame())
end
