Process Add and Remove Actors¶

Warning

Note that actor removals ( Destroy() ) are processed after additions. This means that if an actor is added and removed in a frame, it will not be added to the scene and exist in the next frame. (Although this shouldn’t be possible, since Destroy() requires a reference, if you find a way to add, remove, then add the same actor know that it won’t persist.)
